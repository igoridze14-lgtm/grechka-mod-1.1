# 🐕 Grechka Companion Mod

Fabric мод для Minecraft 1.21.1 который добавляет собаку-спутника Grechka!

## 🎮 Особенности

✅ **Собака ведёт себя как волк** - все механики волка  
✅ **Приручается костью** - кликни костью чтобы приручить  
✅ **Становится спутником** - следует везде за тобой  
✅ **Яйцо призыва** - в креативе или дропается при смерти  
✅ **Уникальная модель и текстура** - твой дизайн!

## 📥 Установка

1. Скачай готовый `grechka-mod-1.0.0.jar` из GitHub Actions
2. Положи в папку `.minecraft/mods/`
3. Запусти Minecraft с Fabric 1.21.1
4. **Готово!** 🐕

## 🚀 Сборка из исходника

```bash
./gradlew build
```

Готовый JAR будет в `build/libs/`

## 🎯 Как использовать в игре

- Ищи собаку как обычное животное
- Возьми **кость** в руку
- Кликни **правой кнопкой** на собаку
- Собака приручена! ❤️
- Она будет следовать за тобой везде

## 📋 Требования

- Minecraft 1.21.1 (Java Edition)
- Fabric Loader 0.15.11+
- Java 21+
